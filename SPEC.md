# 切水果游戏 - 蓝剑切水果

## 项目概述

- **项目名称**: 蓝剑切水果 (Bladesword Fruit Ninja)
- **类型**: HTML5 休闲游戏
- **核心功能**: 在平板上滑动切割飞出的水果，获取分数
- **目标用户**: 小米 Pad 用户休闲娱乐

## 游戏机制

### 操作方式
- 触屏滑动切割水果
- 鼠标滑动（桌面端备用）

### 游戏规则
- 水果从屏幕底部随机抛出
- 切割水果 +10 分
- 连续切割 combo 加成（2连 x1.5, 3连 x2.0, 以此类推）
- 切到炸弹扣 50 分并扣 1 条命
- 漏掉 3 个水果扣 1 条命
- 初始 3 条命

### 水果类型
- 🍎 苹果 - 10分
- 🍌 香蕉 - 10分
- 🍉 西瓜 - 15分
- 🍊 橙子 - 10分
- 🍇 葡萄 - 20分 (较小)
- 🍓 草莓 - 25分 (最小)
- 💣 炸弹 - 扣50分

### 视觉效果
- 水果被切割后分成两半
- 果汁飞溅效果
- 切割轨迹发光
- Combo 文字特效

### 音效设计
- 切割水果: "唰" 音效
- Combo: 连击提示音
- 切到炸弹: 爆炸声
- 过关: 庆祝音效
- 背景音乐: 轻快节奏

## 技术实现

- 单 HTML 文件，内嵌 CSS + JavaScript
- Canvas 2D 渲染
- Web Audio API 音效
- 响应式设计适配平板
